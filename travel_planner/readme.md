# AI Travel Planner

## Objective

Design an AI-based Travel Planner that reuses an existing knowledge base to provide personalized travel recommendations.

## Features

- Tourist Place Recommendation
- Food Recommendation
- Budget Estimation
- Day-wise Travel Itinerary

## Knowledge Base

The system stores information about:

- Goa
- Jaipur
- Kerala

For each destination, the knowledge base contains:

- Recommended duration
- Estimated budget
- Tourist attractions
- Food recommendations


## Output

The system displays:

- Destination details
- Budget estimate
- Tourist attractions
- Food recommendations
- Suggested itinerary

## Conclusion

The AI Travel Planner demonstrates how a knowledge base can be reused to generate personalized travel plans and recommendations.
